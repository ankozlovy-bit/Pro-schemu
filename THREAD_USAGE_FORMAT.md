# Формат файла расхода нитей

## Поддерживаемые форматы

Приложение поддерживает загрузку файлов в формате **TXT** или **RTF**.

## Структура данных

Каждая строка файла должна содержать информацию об одном цвете нитей в следующем формате:

```
КОД_ЦВЕТА НАЗВАНИЕ_ЦВЕТА ДЛИНА_В_МЕТРАХ
```

### Разделители

Поддерживаются следующие разделители:
- Табуляция (`\t`)
- Несколько пробелов подряд
- Одиночные пробелы (с автоматическим определением числовых значений)

## Примеры

### Пример 1: Табуляция

```
14	Pale Apple Green	0.3
28	Medium Light Eggplant	0.2
29	Eggplant	0.3
742	Light tangerine	0.2
745	Banana yellow	0.3
```

### Пример 2: Несколько пробелов

```
471    Tarragon green         125
520    Dark fern green         49
522    Trellis green          850
524    Light grey green       398
597    Iceland blue            55
598    Lagoon turquoise       125
```

### Пример 3: Одиночные пробелы (с автоопределением)

```
310 Black 1.5
311 Navy Blue 2.3
666 Bright Red 0.8
```

## Заголовки (игнорируются)

Следующие строки автоматически пропускаются:
- Строки содержащие "Нить", "Thread", "DMC)"
- Строки содержащие "Описание", "Description"
- Строки содержащие "Всего", "Total"
- Строки с тире (`---`)

## Автоматические расчеты

После загрузки файла система автоматически рассчитывает:

1. **№ п/п** - Порядковый номер (генерируется автоматически)
2. **Номер цвета** - Извлекается из первого столбца файла
3. **Наименование цвета** - Извлекается из среднего столбца (опционально)
4. **Длина (м)** - Берется из файла
5. **Кол-во пасм по 1 м** - Округление длины до целого: `Math.ceil(длина)`
6. **Кол-во мотков** - Расчет на основе 8 метров в мотке: `Math.ceil(длина / 8)`

## Настройки отображения

В интерфейсе доступны следующие настройки:

- **Показать наименования цветов** - включить/выключить столбец с названиями
- **Сообщение под таблицей** - произвольный текст (по умолчанию "Приятной вышивки!")
- **QR-коды** - ссылки на группу ВКонтакте и файл Saga (опционально)

## Формат RTF

При загрузке RTF файлов система автоматически очищает форматирование:
- Удаляет управляющие команды RTF (`\rtf`, `\b`, `\i`, и т.д.)
- Удаляет фигурные скобки
- Нормализует пробелы

## Пример полного файла

```txt
Расход длины нитей (с учетом прибавки 40%)

Номер цвета (DMC)	Наименование цвета	Общая длина (м)
14	Pale Apple Green	0.3
28	Medium Light Eggplant	0.2
29	Eggplant	0.3
742	Light tangerine	0.2
745	Banana yellow	0.3
904	Avocado green	0.7
989	Fennel green	0.4
3747	Pale candy blue	0.3
3823	Ivory	0.5
B5200	Bright white	1.2
```

После обработки этот файл создаст таблицу с 10 строками и автоматически рассчитает количество пасм и мотков для каждого цвета.
