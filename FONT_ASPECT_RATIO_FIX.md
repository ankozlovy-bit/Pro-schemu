# Исправление пропорций шрифтов в PDF ✅

## Проблема

После внедрения Canvas-рендеринга для поддержки кириллицы, текст в PDF был **сплюснут по вертикали**. Шрифты выглядели сжатыми и нечитаемыми.

### Причина:
При добавлении изображений Canvas в PDF использовались **фиксированные размеры** (ширина и высота в мм), которые не соответствовали реальным пропорциям изображения.

```typescript
// ❌ Проблемный код:
const titleImage = renderTextToCanvas("Технический паспорт", { fontSize: 36 });
this.doc.addImage(titleImage, "PNG", x, y, 120, 15); // Фиксированные размеры!
```

## Решение

### 1. Обновлены функции рендеринга

Теперь функции `renderTextToCanvas` и `renderMultilineText` возвращают не только изображение, но и его **реальные размеры**:

```typescript
interface CanvasImageData {
  dataUrl: string;  // Base64 PNG
  width: number;    // Ширина в пикселях
  height: number;   // Высота в пикселях
}

function renderTextToCanvas(text: string, options): CanvasImageData {
  // Создание canvas с правильными размерами
  const canvas = document.createElement("canvas");
  const ctx = canvas.getContext("2d")!;
  
  // Использование 2x DPI для лучшего качества
  const dpi = 2;
  const scaledFontSize = fontSize * dpi;
  
  // Измерение и установка размеров
  ctx.font = `${fontWeight} ${scaledFontSize}px ${fontFamily}`;
  const metrics = ctx.measureText(text);
  canvas.width = metrics.width + padding;
  canvas.height = scaledFontSize * 1.4 + padding;
  
  // Рендеринг текста
  ctx.fillText(text, x, y);
  
  // Возврат размеров с учетом DPI
  return {
    dataUrl: canvas.toDataURL("image/png"),
    width: canvas.width / dpi,  // Реальная ширина
    height: canvas.height / dpi, // Реальная высота
  };
}
```

### 2. Правильное масштабирование

При добавлении изображения в PDF используется **коэффициент преобразования пикселей в миллиметры**:

```typescript
// ✅ Правильный код:
const titleImage = renderTextToCanvas("Технический паспорт", { fontSize: 36 });

// Масштаб: пиксели → миллиметры (при 96 DPI ≈ 0.26 мм/px)
const scale = 0.26;
const widthMM = titleImage.width * scale;
const heightMM = titleImage.height * scale;

// Добавление с сохранением пропорций
this.doc.addImage(
  titleImage.dataUrl, 
  "PNG", 
  x, 
  y, 
  widthMM,   // Правильная ширина
  heightMM,  // Правильная высота
  undefined, 
  "FAST"
);
```

### 3. Улучшено качество

- **DPI увеличен до 2x** - текст более чёткий
- **Padding увеличен** - текст не обрезается
- **Точное измерение** - используется `measureText()` API

## Технические детали

### Коэффициент масштабирования:

```
1 пиксель ≈ 0.2645833333 мм (при 96 DPI)
Округлено до 0.26 для простоты
```

### Пример преобразования:

```typescript
Canvas: 400px × 60px (2x DPI: 800px × 120px)
      ↓ (делим на DPI)
Логические размеры: 400px × 60px
      ↓ (умножаем на 0.26)
PDF размеры: 104mm × 15.6mm
```
