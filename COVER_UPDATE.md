# ✅ Обновление структуры обложки PDF

## Что изменилось

### 1. Новая структура текста

Обложка теперь следует профессиональному формату:

```
┌─────────────────────────────────────┐
│                                     │
│    Схема для вышивки крестом       │  ← Заголовок категории
│                                     │
│         «Название схемы»            │  ← Название в кавычках
│                                     │
│       ┌───────────────┐             │
│       │               │             │
│       │   Превью      │             │  ← Изображение вышивки
│       │   вышивки     │             │
│       │               │             │
│       └───────────────┘             │
│                                     │
│    Дизайнер: Имя дизайнера         │  ← Информация о дизайнере
│                                     │
│                                     │
│         StitchPDF                   │  ← Логотип внизу
└─────────────────────────────────────┘
```

### 2. Новые возможности

#### Превью изображения вышивки
- **Отдельная загрузка** превью схемы
- Отображается в центре обложки
- Рамка с белой обводкой
- Размер: 100×100 мм в PDF

#### Фоновое изображение (опционально)
- Декоративный фон обложки
- Отображается с низкой прозрачностью (15%)
- Не перекрывает основной контент

### 3. Многоязычность

Текст адаптируется согласно выбранному языку:

| Элемент | Русский (RU) | English (EN) |
|---------|-------------|--------------|
| Заголовок | Схема для вышивки крестом | Cross Stitch Pattern |
| Дизайнер | Дизайнер: | Designer: |
| Кавычки | «Название» | «Name» |

## Технические изменения

### 1. Обновлённые типы данных

```typescript
interface ProjectData {
  // ...
  coverImage?: string;        // Фоновое изображение (декоративное)
  coverPreviewImage?: string; // Превью вышивки (основное)
  coverTitle?: string;        // Название схемы
  coverDesigner?: string;     // Имя дизайнера
  coverBackgroundColor?: string; // Цвет фона
}
```

### 2. Обновлённый CoverEditor

#### Новые пропсы:
- `coverPreviewImage?: string` - превью изображение
- `onPreviewImageUpload: (imageData: string) => void` - обработчик загрузки

#### Новый UI в табе "Дизайн":

**1. Превью вышивки** (основное)
- Drag & drop зона
- Подпись: "Изображение вашей схемы для отображения на обложке"
- Приоритетное расположение

**2. Фоновое изображение** (опционально)
- Drag & drop зона
- Подпись: "Декоративное изображение для фона обложки"
- Второстепенное расположение

**3. Выбор цвета фона**
- Палитра из 8 пресетов
- Цветовой пикер
- Ручной ввод HEX

### 3. Обновлённый PDF генератор

#### Новая функция `addCoverPage()`:

```typescript
private async addCoverPage(): Promise<void> {
  // 1. Фон с выбранным цветом
  // 2. Декоративное фоновое изображение (15% opacity)
  // 3. Декоративные точки
  
  // 4. Заголовок: "Схема для вышивки крестом"
  //    Шрифт: Georgia, 20px
  
  // 5. Название в кавычках: «Название схемы»
  //    Шрифт: Georgia Bold, 36px
  
  // 6. Превью изображение (если загружено)
  //    Размер: 100×100 мм
  //    С белой рамкой
  
  // 7. Дизайнер: Имя
  //    Шрифт: Arial, 18px
  
  // 8. Логотип StitchPDF внизу
  //    Шрифт: Arial, 14px
}
```

#### Позиционирование:
- **Гибкая вертикальная разметка** (yPos)
- Автоматическое центрирование по горизонтали
- Адаптивные отступы между элементами

## Использование

### Шаг 1: Перейдите на Шаг 2

### Шаг 2: Заполните текстовую информацию (таб "Текст")
- **Заголовок обложки:** Название вашей схемы
- **Подзаголовок:** Оставьте пустым (не используется в новой структуре)
- **Дизайнер:** Ваше имя или студия

### Шаг 3: Загрузите изображения (таб "Дизайн")

#### A. Превью вышивки (рекомендуется)
1. Нажмите "Выбрать файл" в секции "Превью вышивки"
2. Выберите изображение готовой вышивки или рендер схемы
3. Рекомендуемый размер: 800×800 px или больше
4. Формат: JPG, PNG

**Пример:** Фотография готовой вышивки в рамке

#### B. Фоновое изображение (опционально)
1. Нажмите "Выбрать файл" в секции "Фоновое изображение"
2. Выберите декоративное изображение
3. Будет отображаться полупрозрачным на фоне

**Пример:** Текстура ткани, абстрактный узор

#### C. Цвет фона
1. Выберите из палитры предустановленных цветов
2. Или используйте цветовой пикер
3. Или введите HEX код вручную

**Рекомендуемые цвета:**
- Soft Sage Green (#E8F0E8) - нежный зелёный
- Dusty Rose (#F4E7E7) - пыльная роза
- Warm Off-White (#FCFAF9) - тёплый белый

### Шаг 4: Просмотр в реальном времени

Превью справа показывает как будет выглядеть обложка в PDF:
- ✅ Правильная структура текста
- ✅ Кавычки вокруг названия
- ✅ Превью изображение (если загружено)
- ✅ Имя дизайнера
- ✅ Логотип StitchPDF

## Рекомендации по дизайну

### Для превью изображения:
- ✅ Квадратное соотношение (1:1)
- ✅ Высокое разрешение (минимум 800×800 px)
- ✅ Хороший контраст с фоном
- ✅ Центрированная композиция
- ❌ Избегайте вытянутых изображений
- ❌ Избегайте низкого разрешения (< 500 px)

### Для фона:
- ✅ Светлые пастельные цвета
- ✅ Минимальный контраст (фон не должен отвлекать)
- ✅ Однотонные или с лёгким паттерном
- ❌ Избегайте тёмных цветов (текст может быть не читаемым)
- ❌ Избегайте ярких, насыщенных цветов

### Для названия:
- ✅ Краткое и ёмкое (2-5 слов)
- ✅ Без лишних символов
- ✅ Правильная орфография
- ❌ Избегайте длинных названий (> 40 символов)
- ❌ Избегайте специальных символов

## Технические детали

### Размеры в PDF:

| Элемент | Шрифт | Размер в px | Размер в PDF |
|---------|-------|-------------|--------------|
| Заголовок категории | Georgia 20px | 20 | ~5-6 mm |
| Название в кавычках | Georgia Bold 36px | 36 | ~9-12 mm |
| Превью изображение | - | - | 100×100 mm |
| Дизайнер | Arial 18px | 18 | ~4-5 mm |
| Логотип | Arial 14px | 14 | ~3-4 mm |

### Позиционирование:

```typescript
let yPos = MARGIN + 20;  // Начало (35 mm от верха)

yPos += headerHeightMM + 10;   // После заголовка категории
yPos += titleHeightMM + 15;    // После названия
yPos += 100 + 15;              // После превью (если есть)
// Дизайнер на yPos
// Логотип на A4_HEIGHT - 20 (фиксированно внизу)
```
