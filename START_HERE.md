## 🎯 Основной функционал

### 6-шаговый Workflow

1. **Загрузка схемы**
   - Импорт XSP файлов (бинарные)
   - Автоматический парсинг цветов и параметров

2. **Обложка**
   - Визуальный редактор
   - Загрузка готовой обложки (PDF/JPG/PNG)
   - Настройка стиля (углы, превью, шрифты)

3. **Ключ схемы**
   - Загрузка готового ключа (PDF)
   - **Генератор ключа** (до 5 языков одновременно!)

4. **Расход мулине**
   - Загрузка RTF/TXT файлов
   - Калькулятор расхода
   - **QR-коды** (библиотека + пользовательские)

5. **Загрузка схем**
   - Множественная загрузка PDF схем
   - Управление списком

6. **Финальный PDF**
   - **Drag & Drop** порядка документов
   - Селективное включение разделов
   - Генерация итогового PDF

- XSP (бинарные) - схемы Cross Stitch Professional Platinum

## 💡 Советы по использованию

### Для быстрой работы:
1. Подготовьте все файлы заранее (XSP, PDF схем)
2. Используйте библиотеку QR-кодов (Шаг 4)
3. Сохраняйте порядок документов (Шаг 6)

### Для качественного результата:
1. Используйте высококачественные изображения для обложки
2. Проверяйте превью на каждом шаге
3. Настройте переводы заголовков (если нужно)

### Для мультиязычности:
1. Используйте генератор ключа на Шаге 3
2. Выберите до 5 языков одновременно
3. Экспортируйте RTF для вставки в документ
