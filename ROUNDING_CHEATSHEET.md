# 🔄 Шпаргалка по округлению .05

## Быстрое правило

```
Если (Длина ÷ 2) заканчивается на .05
→ Округлить до .1
```

---

## 📊 Таблица

| Длина | ÷ 2 | Округление | Итого |
|-------|-----|------------|-------|
| 0.1   | 0.05 | **→ 0.1** | 0.1 ✅ |
| 2.1   | 1.05 | **→ 1.1** | 1.1 ✅ |
| 4.1   | 2.05 | **→ 2.1** | 2.1 ✅ |
| 6.1   | 3.05 | **→ 3.1** | 3.1 ✅ |
| 8.1   | 4.05 | **→ 4.1** | 4.1 ✅ |
| 10.1  | 5.05 | **→ 5.1** | 5.1 ✅ |
| 0.3   | 0.15 | -         | 0.15 ✅ |
| 2.5   | 1.25 | -         | 1.25 ✅ |
| 4.8   | 2.4  | -         | 2.4 ✅ |

---

## 🔍 Как определить .05?

### С округлением:
- 0.**05** ✅
- 1.**05** ✅
- 2.**05** ✅
- 3.**05** ✅
- 10.**05** ✅

### Без округления:
- 0.**15** ❌
- 1.**25** ❌
- 2.**4** ❌
- 3.**85** ❌

---

## 💡 Быстрые примеры

### Пример 1:
```
666+703 = 0.1м
0.1 ÷ 2 = 0.05 → 0.1
Каждому: +0.1м ✅
```

### Пример 2:
```
310+3799 = 2.1м
2.1 ÷ 2 = 1.05 → 1.1
Каждому: +1.1м ✅
```

### Пример 3:
```
666+703 = 0.3м
0.3 ÷ 2 = 0.15
Каждому: +0.15м (без округления) ✅
```

---

## 🔔 Как увидеть в консоли

### С округлением:
```
🔄 Rounded 0.05 → 0.1 (ends in .05)
```

### Без округления:
```
(лога нет)
```

---

## 📐 Формула

```javascript
// Проверка
decimal = число - целая_часть
if (decimal ≈ 0.05):
  округлить вверх до .1
else:
  оставить как есть
```

---

## ✅ Когда применяется

- ✅ **ТОЛЬКО** при делении смешанных цветов
- ✅ **ТОЛЬКО** если результат заканчивается на .05
- ✅ **ВСЕГДА** в большую сторону (до .1)

---

## ❌ Когда НЕ применяется

- ❌ К обычным (не смешанным) цветам
- ❌ К числам, НЕ заканчивающимся на .05
- ❌ К исходной длине (только к половине)
