## ✅ Что Реализовано

### 🎨 Новая Функция: Автоматическое Распознавание Комбинированных Цветов

Приложение **ПроСхему** теперь умеет автоматически обрабатывать комбинированные (смешанные) цвета DMC в файлах расхода нитей!

## 🚀 Что Это Значит?

Ваши друзья могут загружать файлы с любыми вариантами записи комбинированных цветов:

### Примеры Обработки:

| Что в файле | Что получается | Магия! |
|-------------|----------------|--------|
| `3103799` | `310` + `3799` | ✨ Автоматическое разделение склеенных номеров |
| `666705` | `666` + `705` | ✨ Распознавание без разделителя |
| `3+3799` | `310` + `3799` | ✨ Исправление неполного номера |
| `666+705` | `666` + `705` | ✨ Корректная обработка |

### Как Это Работает:

1. **Пользователь загружает файл** с расходом нитей
2. **Система автоматически находит** комбинированные цвета
3. **Длина делится пополам** между двумя цветами
4. **Половины добавляются** к соответствующим цветам
5. **Строки с "+"" удаляются** из финальной таблицы
6. **Результат сортируется** по возрастанию номеров

## 📋 Пример

### Входной файл:
```
Номер цвета (DMC)	Общая длина (м)
666	                2.2
3799	            4.0
3103799	            2.5
3+3799	            0.8
666+705	            0.3
```

### Результат в приложении:
```
№  DMC   Длина (м)
1  310   1.65      ← Создан из комбинаций (1.25 + 0.4)
2  666   2.35      ← Исходный (2.2) + часть от 666+705 (0.15)
3  705   0.15      ← Создан из 666+705
4  3799  5.65      ← Исходный (4.0) + части (1.25 + 0.4)
```
