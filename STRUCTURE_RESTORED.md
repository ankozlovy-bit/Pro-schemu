# ПроСхему - Структура приложения восстановлена ✅

## Текущее состояние (13.03.2026)

Приложение успешно восстановлено до полной функциональности после исправления ошибок компиляции.

## Архитектура приложения

### 🏠 Главный компонент
- **`/src/app/App.tsx`** - корневой компонент приложения с навигацией между страницами
- Использует простое состояние для переключения страниц без React Router
- Добавлен `Toaster` для уведомлений (sonner)

### 📄 Страницы

1. **WelcomePage** (`/src/app/pages/WelcomePage.tsx`)
   - Главная страница с героем
   - Навигация к основным функциям приложения
   - Дизайн "Ethereal Tech" с палитрой "Soft Botanical"

2. **MainWorkflow** (`/src/app/pages/MainWorkflow.tsx`)
   - Полный workflow создания PDF схемы (6 шагов)
   - Прогресс-бар с визуализацией шагов
   - ScrollProgress для отслеживания прокрутки

3. **KeyGeneratorPage** (`/src/app/pages/KeyGeneratorPage.tsx`)
   - Генератор ключа схемы в RTF формате
   - Поддержка 6 языков (русский, английский, немецкий, французский, испанский, итальянский)
   - 5 стильных пресетов для оформления

4. **ProjectsHistoryPage** (`/src/app/pages/ProjectsHistoryPage.tsx`)
   - История созданных PDF схем
   - Пока показывает пустое состояние (будет реализовано с localStorage)

5. **SettingsPage** (`/src/app/pages/SettingsPage.tsx`)
   - Настройки профиля дизайнера
   - Настройки оформления
   - Настройки экспорта PDF
   - Выбор языка интерфейса

### 🔄 Workflow - 6 шагов

#### Шаг 1: Загрузка схемы (`StepOne.tsx`)
- Ввод имени дизайнера
- Ввод названия схемы
- Выбор языка (RU/EN)
- Выбор каунта канвы
- Выбор цвета ткани

#### Шаг 2: Обложка (`StepTwo.tsx`)
- **CoverEditor** - редактор обложки
- Загрузка фонового изображения
- Загрузка превью вышивки
- Настройка текстов (заголовок, подзаголовок, дизайнер)
- Выбор цветов фона и текста
- Генерация PDF обложки

#### Шаг 3: Ключ схемы (`StepThree.tsx`)
- Загрузка готового PDF с ключом схемы
- **LanguageSelector** - выбор до 5 языков для генератора
- Кнопка открытия генератора ключа схемы
- Checkbox включения в финальный PDF

#### Шаг 4: Расход мулине (`StepFour.tsx`)
- **ThreadCalculator** - калькулятор расхода ниток
- Загрузка RTF/TXT файла с данными
- Парсинг Excel/RTF форматов
- Настройка отображения (названия цветов)
- Добавление кастомного сообщения
- Добавление QR-кодов
- Выбор шрифта для таблицы
- Генерация PDF с таблицей расхода

#### Шаг 5: Загрузка схем (`StepFive.tsx`)
- Загрузка до 4 PDF файлов со схемами
- Drag & Drop поддержка
- Предпросмотр загруженных файлов
- Управление порядком PDF файлов

#### Шаг 6: Финальный PDF (`StepSix.tsx`)
- Сводка проекта
- Предпросмотр всех разделов
- Генерация финального PDF документа
- Скачивание готового файла

### 🧩 Основные компоненты

#### Редакторы
- **CoverEditor** - редактор обложки с live preview
- **ThreadCalculator** - калькулятор расхода мулине
- **KeyGenerator** - генератор ключа схемы (RTF)

#### UI компоненты
- **LanguageSelector** - выбор и сортировка языков
- **ColorPicker** - выбор цвета с пресетами
- **AdvancedColorPicker** - расширенный колорпикер
- **ScrollProgress** - прогресс-бар прокрутки
- **LoadingSpinner** - индикатор загрузки

#### Утилиты и сервисы
- **pdfGenerator.ts** - генерация PDF (jsPDF)
- **pdfMerger.ts** - объединение PDF файлов
- **rtf-generator.ts** - генерация RTF файлов
- **rtfParser.ts** - парсинг RTF файлов
- **binaryXspParser.ts** - парсинг XSP файлов
- **threadCalculatorPDF.ts** - PDF для расхода ниток
- **keyGeneratorTranslations.ts** - переводы для 6 языков
