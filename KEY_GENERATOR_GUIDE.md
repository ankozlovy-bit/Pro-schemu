# Руководство по Генератору Ключа Схемы

## Обзор

Генератор ключа схемы - это отдельная функция приложения StitchPDF для создания RTF-файлов с переведёнными заголовками таблиц для Cross Stitch Professional Platinum.

## Особенности

- ✅ Загрузка RTF файлов с ключами схемы
- ✅ Автоматическое извлечение заголовков таблиц и разделов
- ✅ Поддержка 6 языков: Русский, English, Deutsch, Français, Español, Italiano
- ✅ Возможность ручного редактирования переводов
- ✅ Сохранение всей структуры RTF файла
- ✅ Красивый дизайн в стиле Ethereal Tech

## Как использовать

### Шаг 1: Открытие генератора

В главном меню приложения нажмите кнопку **"Генератор ключа"** (иконка FileKey) в правом верхнем углу.

### Шаг 2: Загрузка RTF файла

1. Перетащите RTF файл в область загрузки
2. Или нажмите кнопку **"Выбрать RTF файл"**
3. Приложение автоматически извлечёт заголовки

### Шаг 3: Выбор языка

1. В выпадающем списке выберите целевой язык перевода
2. Заголовки автоматически заполнятся переводами

### Шаг 4: Настройка переводов

1. Переключайтесь между вкладками **"Заголовки таблицы"** и **"Разделы"**
2. При необходимости отредактируйте переводы вручную
3. Каждый заголовок можно изменить индивидуально

### Шаг 5: Скачивание

Нажмите кнопку **"Скачать переведённый RTF"** для экспорта файла.

## Поддерживаемые заголовки

### Заголовки таблицы
- Цвет / Color / Farbe / Couleur
- Символ / Symbol / Symbol / Symbole
- № цвета / Thread # / Fadennr. / N° de fil
- Нити / Strands / Fäden / Brins

### Разделы
- Полукрест / Half Cross / Halbkreuz / Demi-point
- Бэкстич (шов «назад иголку») / Backstitch / Rückstich / Point arrière
- Французкий узелок / French Knot / Französischer Knoten / Point de nœud

## Технические детали

### Формат файла
- Поддерживается только формат RTF
- Кодировка определяется автоматически
- Сохраняется вся структура RTF (таблицы, форматирование)

### Парсинг заголовков
- Автоматическое извлечение заголовков из RTF
- Распознавание кириллицы и латиницы
- Поддержка специальных символов (№, «», ())

### Генерация файла
- Замена только текста заголовков
- Сохранение всех RTF-команд форматирования
- Поддержка Unicode-кодировки

## Примечания

- RTF файл должен содержать таблицы с заголовками
- Для корректной работы рекомендуется использовать файлы из Cross Stitch Professional Platinum
- Все изменения сохраняются только в новом файле, оригинал остаётся нетронутым

## Известные ограничения

- Поддерживается только формат RTF (не DOCX, PDF)
- Обнаружение заголовков основано на ключевых словах
- Сложные форматирования могут требовать дополнительной проверки

## Troubleshooting

### Заголовки не обнаружены
- Убедитесь, что файл содержит таблицы
- Проверьте, что заголовки написаны на русском языке
- Попробуйте добавить переводы вручную

### Файл не скачивается
- Проверьте настройки браузера для скачивания файлов
- Убедитесь, что у вас есть права на запись в папку загрузок

### Некорректные переводы
- Используйте ручное редактирование переводов
- Проверьте соответствие оригинальных заголовков
