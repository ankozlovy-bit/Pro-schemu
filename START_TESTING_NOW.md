# 🚀 НАЧАТЬ ТЕСТИРОВАНИЕ СЕЙЧАС!

## 🎯 5 Простых Шагов

### 1️⃣ Откройте Приложение
```
Запустите ПроСхему в браузере
```

### 2️⃣ Перейдите на Шаг 5
```
Кликните "Шаг 5: Расход нитей"
```

### 3️⃣ Загрузите Тестовый Файл
```
Файл: /public/thread-usage-test-blended.txt

Или создайте свой .txt файл:
  Номер цвета (DMC)	Общая длина (м)
  666	                2.2
  3799	            4.0
  3103799	            2.5
  3+3799	            0.8
  666+705	            0.3
```

### 4️⃣ Откройте Консоль
```
Нажмите F12 (или Ctrl+Shift+I)
Перейдите на вкладку "Console"
```

### 5️⃣ Проверьте Результат
```
✅ В таблице должны быть:
   - 310 (1.65м) ← создан из комбинаций
   - 666 (2.35м) ← обновлён
   - 705 (0.15м) ← создан
   - 3799 (5.65м) ← обновлён

✅ В консоли должны быть логи с эмодзи:
   🎨 🔧 🎯 ✂️ ➕ 🆕 🗑️ 📋
```

## 🎨 Примеры Обработки

### Пример 1: Склеенные номера
```
Вход:  3103799  (2.5м)
Выход: 310 (1.25м) + 3799 (1.25м)
```

### Пример 2: Неполный номер
```
Вход:  3+3799  (0.8м)
Выход: 310 (0.4м) + 3799 (0.4м)
```

### Пример 3: Корректная комбинация
```
Вход:  666+705  (0.3м)
Выход: 666 (0.15м) + 705 (0.15м)
```

### Пример 4: Объединение дубликатов
```
Вход:  
  310+3799  (2.5м)
  310+3799  (0.8м)
  
Обработка:
  Объединение: 2.5м + 0.8м = 3.3м
  Деление: 3.3м ÷ 2 = 1.65м
  
Выход:
  310:  1.65м
  3799: 1.65м (+ исходные 4.0м = 5.65м)
```

## 🐛 Если Что-то Не Работает

### Проблема: Файл не загружается
```
Решение:
- Проверьте формат (.txt)
- Используйте табуляцию между колонками
- Убедитесь что есть заголовки
```

### Проблема: Комбинации не распознаются
```
Решение:
- Откройте консоль (F12)
- Найдите логи с 🎨
- Проверьте, что коды только из цифр и "+"
```

### Проблема: Длины неправильные
```
Решение:
- Проверьте логи с ✂️
- Убедитесь что деление на 2
- Проверьте логи с ➕
```

### Проблема: Не видно логов
```
Решение:
- Откройте консоль (F12)
- Очистите консоль (Ctrl+L)
- Перезагрузите файл
```
