# StitchPDF - Структура приложения

## Обзор

StitchPDF — это профессиональное веб-приложение для дизайнеров вышивки с двумя основными функциями:

1. **Создание PDF схем** - полный многошаговый workflow для генерации профессиональных PDF схем вышивки формата A4
2. **Генератор ключа схемы** - инструмент для создания RTF-файлов с переводом заголовков таблиц для импорта в Cross Stitch Professional Platinum

## Дизайн-система: Ethereal Tech

### Цветовая палитра
- **Soft Sage Green**: `#E8F0E8` - акценты, кнопки, градиенты
- **Dusty Rose**: `#F4E7E7` - вторичные акценты, мягкие фоны
- **Warm Off-White**: `#FCFAF9` - основной фон
- **Deep Charcoal**: `#2D2D2D` - основной текст, кнопки

### Типографика
- **Заголовки**: Playfair Display (serif)
- **Основной текст**: Inter (sans-serif)

### Стилистические особенности
- **Glassmorphism**: `backdrop-blur-xl`, полупрозрачные фоны
- **Скругленные углы**: 24px+ (`rounded-[24px]`, `rounded-[32px]`)
- **Анимации**: Motion (Framer Motion) для плавных переходов
- **Адаптивность**: Полная поддержка mobile и desktop

## Роутинг

### Страницы

| Путь               | Компонент          | Описание                                    |
|--------------------|--------------------|---------------------------------------------|
| `/`                | `Home`             | Главная страница с выбором функций          |
| `/workflow`        | `MainWorkflow`     | Многошаговый workflow создания PDF          |
| `/key-generator`   | `KeyGeneratorPage` | Генератор RTF ключа схемы                   |
| `*`                | `NotFound`         | 404 страница                                |

## Workflow создания PDF (5 шагов)

### Шаг 1: Загрузка схемы
- Загрузка XSP файла (Cross Stitch Professional)
- Автоматический парсинг размеров и данных схемы
- Опциональная загрузка шрифта

### Шаг 2: Обложка
- Создание дизайнерской обложки
- Загрузка фонового изображения
- Загрузка превью вышивки
- Настройка текста (название, автор)
- Выбор цветов и стиля

### Шаг 3: Ключ схемы
- Загрузка готового PDF ключа символов
- Опция включения/выключения в финальный PDF

### Шаг 4: Расход мулине
- Загрузка или создание таблицы расхода ниток
- Настройка отображения (названия цветов, шрифты)
- Добавление QR-кодов и сообщений

### Шаг 5: Финальный PDF
- Превью всех страниц
- Выбор страниц для включения
- Экспорт итогового PDF файла

## Генератор ключа схемы

### Функциональность
- Загрузка RTF файла с ключом схемы (русский язык)
- Автоматическое извлечение заголовков таблиц и разделов
- Выбор языка перевода (Английский, Немецкий, Французский, Испанский, Итальянский)
- Ручная настройка переводов
- Генерация нового RTF файла с переведёнными заголовками

### Поддерживаемые языки
- Русский (исходный)
- English
- Deutsch
- Français
- Español
- Italiano
