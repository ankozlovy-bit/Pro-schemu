# Тестирование смешанных цветов DMC - Финальная версия ✅

## Что было исправлено

### 1. ✅ Нормализация склеенных номеров
- **"3103799"** → **"310+3799"** (автоматическое разделение)
- **"666703"** → **"666+703"** (6-значные числа)
- **"3103799"** → **"310+3799"** (7-значные числа)

### 2. ✅ Нормализация неполных записей
- **"3+3799"** → **"310+3799"** (дополнение ведущими нулями)
- **"8+999"** → **"008+999"**

### 3. ✅ Корректное распределение длины
- Длина всегда делится на **2** (не на количество цветов)
- Половина добавляется к первому цвету
- Половина добавляется ко второму цвету
- Пример: **"310+3799" с 2.5м** → **310: +1.25м, 3799: +1.25м**

### 4. ✅ Скрытие смешанных строк из таблицы
- Строки с **"+"** в первом столбце **не отображаются** в финальной таблице
- После обработки они удаляются из `groupedMap`

## Примеры тестовых данных

### Пример 1: Склеенные номера
```
Номер цвета (DMC)    Наименование цвета    Общая длина (м)
310                  Anthracite grey       2.0
3799                 Charcoal grey         3.0
3103799              -                     2.5
```

**Ожидаемый результат:**
- Строка "3103799" НЕ показывается
- 310: 2.0 + 1.25 = **3.25м**
- 3799: 3.0 + 1.25 = **4.25м**

### Пример 2: Неполная запись с "+"
```
Номер цвета (DMC)    Наименование цвета    Общая длина (м)
310                  Anthracite grey       4.0
3799                 Charcoal grey         2.1
3+3799               -                     0.8
```

**Ожидаемый результат:**
- Строка "3+3799" НЕ показывается (нормализуется в "310+3799")
- 310: 4.0 + 0.4 = **4.4м**
- 3799: 2.1 + 0.4 = **2.5м**

### Пример 3: Обычная запись с "+"
```
Номер цвета (DMC)    Наименование цвета    Общая длина (м)
666                  Christmas red         1.5
703                  Chartreuse            2.0
666+703              -                     0.3
```

**Ожидаемый результат:**
- Строка "666+703" НЕ показывается
- 666: 1.5 + 0.15 = **1.65м**
- 703: 2.0 + 0.15 = **2.15м**

## Как проверить

1. **Откройте приложение** в браузере
2. **Перейдите на Шаг 3** (Загрузка ключа)
3. **Загрузите файл** с тестовыми данными
4. **Откройте консоль** браузера (F12)
5. **Проверьте логи** - должны совпадать с примерами выше
6. **Проверьте таблицу** - строки с "+" не должны отображаться
7. **Проверьте значения** - длины должны быть правильно распределены

- [ ] Длина делится на 2 и правильно распределяется
- [ ] Строки со знаком "+" НЕ отображаются в таблице

## Примечания

- Функция `normalizeBlendedColor()` вызывается **после парсинга** каждой строки
- Обработка смешанных цветов происходит **после группировки** дубликатов
- Удаление смешанных строк происходит **перед сортировкой** финальной таблицы
- Логирование помогает отследить каждый шаг трансформации данных
