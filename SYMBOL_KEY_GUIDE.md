# 📋 Руководство: Генерация "Ключа для дизайна"

## Что это?

**Ключ для дизайна (Symbol Key)** - это профессиональный документ с таблицами символов, DMC кодов и инструкций для вышивальщиц. Генерируется автоматически из XSP файлов.

---

## 🚀 Быстрый старт (3 шага)

### Шаг 1: Загрузите XSP файл
1. Откройте **Шаг 1: Загрузка XSP**
2. Перетащите ваш `.xsp` файл (формат XSPPLAT)
3. Нажмите **"Продолжить"**

### Шаг 2: Создайте обложку
1. Заполните название дизайна
2. Укажите имя дизайнера
3. Нажмите **"Продолжить"**

### Шаг 3: Сформируйте ключ
1. Нажмите **"Анализировать"** в блоке "Отладка XSP файла"
2. Проверьте распознанные данные в зеленой карточке:
   - ✅ Название
   - ✅ Размер схемы
   - ✅ Палитра DMC
   - ✅ Количество цветов

3. Прокрутите вниз до раздела **"Ключ для дизайна"**
4. Нажмите **"Скачать HTML"** или **"Печать"**

✅ **Готово!** У вас есть профессиональный документ с таблицами.

---

## 📊 Что включено в документ?

### Заголовок:
- **Название**: "Ключ для дизайна / Instructions and Symbol Key"
- **Дизайн**: Название вашей схемы
- **Дизайнер**: Имя автора
- **Ссылки**: На сообщества (если указаны в настройках)

### Информация о схеме:
- **Канва**: 14/16/18 каунта (зависит от настроек)
- **Размер**: Ширина × Высота стежков
- **Палитра**: DMC или Anchor

### Таблицы:
1. **Крест / Cross stitch** (двойная колонка)
   - Цвет (квадратик)
   - Символ
   - № цвета (DMC)
   - Количество нитей

2. **Полукрест / Half Cross Stitches**
3. **Бэкстич / Backstitch**
4. **Французский узелок / French Knots**

### Примечание:
⚠️ Предупреждение о заштрихованных блоках (на русском и английском)

## 🎨 Поддерживаемые форматы

### Бинарный XSPPLAT ✅
- Pattern Maker
- PC Stitch (некоторые версии)
- **Сигнатура**: `XSPPLAT` в начале файла

### Что извлекается автоматически:
- ✅ Название дизайна
- ✅ Имя автора
- ✅ Размер схемы (ширина × высота)
- ✅ DMC коды (до 50 цветов)
- ✅ Hex цвета (из базы 100+ DMC)
- ✅ Количество нитей (по умолчанию 2)

## 🔧 Отладка

### Если данные не распознались:

1. **Откройте консоль браузера** (F12)
2. Нажмите **"Анализировать"**
3. Посмотрите логи:
   ```
   🔍 Обнаружена сигнатура Pattern Maker: XSPPLAT
   📝 Найденные строки: [...]
   🎨 Найдено DMC кодов: 15
   📊 Результаты парсинга Pattern Maker:
     Название: Морозная элегия
     Автор: Наталья Козина
     Размер: 156 x 157
     Цветов: 15
   ```

4. **Проверьте "Сырые данные"**:
   - Должно начинаться с `XSPPLAT`
   - Должны быть видны DMC коды (числа 100-9999)

### Если цвета неточные:

**Вариант 1**: Редактируйте вручную
- На шаге 3 вы можете изменить:
  - Символы
  - DMC коды
  - Количество нитей
  - Видимость цветов

**Вариант 2**: Экспортируйте в XML
- Откройте XSP в Pattern Maker
- File → Export → XML format
- Загрузите XML версию (будущая поддержка)

## 💡 Советы и трюки

### 1. Настройте глобальные параметры
Нажмите **⚙️ Настройки** вверху:
- Имя дизайнера (RU/EN)
- QR-код группы
- QR-код Telegram канала

→ Эти данные автоматически вставятся в документ!

### 2. Используйте двухколоночный формат
Для крестика используется двойная таблица:
```
Цвет | Символ | № | Нити || Цвет | Символ | № | Нити
```
→ Экономит место на странице!

### 3. Печать на A4
Документ оптимизирован для печати:
- Формат: A4 (210 × 297 мм)
- Поля: 20 мм
- Шрифт: читаемый на печати

## ⚡ Частые вопросы (FAQ)

### В: Почему не все цвета распознались?
**О**: Бинарный формат не всегда содержит полную палитру. Используйте отладчик для проверки.

### В: Можно ли изменить символы?
**О**: Да! На шаге 3 в таблицах вы можете менять символы, DMC коды и количество нитей.

### В: Поддерживается ли Anchor?
**О**: Сейчас основной фокус на DMC. Anchor - в планах.

### В: Как добавить свой логотип?
**О**: Используйте настройки (⚙️) для загрузки QR-кодов. Логотипы - в разработке.

### В: Файл слишком большой?
**О**: HTML файлы обычно 20-50 KB. Если больше - напишите в поддержку.
