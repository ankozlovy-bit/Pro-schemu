# 🔄 Примеры округления чисел с .05

## Правило округления

**Если при делении длины пополам получается число, заканчивающееся на .05, оно округляется в бОльшую сторону (до .1)**

---

## 📊 Примеры

### Пример 1: 0.1м ÷ 2 = 0.05м
**Входные данные:**
```
666      Christmas red      1.0
703      Chartreuse         2.0
666+703  -                  0.1
```

**Обработка:**
```
0.1м ÷ 2 = 0.05м
0.05м → 0.1м (округление вверх) 🔄
```

**Результат:**
- `666`: 1.0 + **0.1** = **1.1м** ✅
- `703`: 2.0 + **0.1** = **2.1м** ✅

**Консоль:**
```
🔀 Splitting blended color "666+703" (0.1m) between first 2 colors
  🔄 Rounded 0.05 → 0.1 (ends in .05)
  ➗ Length per component: 0.1m each
```

## 💡 Почему это важно?

### Проблема без округления:
```
Смешанный цвет: 0.1м
Деление: 0.1 ÷ 2 = 0.05м на компонент
Итого: 310 получает +0.05м, 3799 получает +0.05м

❌ 0.05м - это очень малая длина для практического использования
```

### Решение с округлением:
```
Смешанный цвет: 0.1м
Деление: 0.1 ÷ 2 = 0.05м → округляем до 0.1м
Итого: 310 получает +0.1м, 3799 получает +0.1м

✅ 0.1м - более практичная длина для работы
```

## 🧮 Математическая логика

```javascript
// Функция округления
const roundUpIfNeeded = (value: number): number => {
  // Проверяем, заканчивается ли число на .05
  const decimalPart = value - Math.floor(value);
  const isPointZeroFive = Math.abs(decimalPart - 0.05) < 0.001;
  
  if (isPointZeroFive) {
    // Округляем вверх до .1
    const rounded = Math.ceil(value * 10) / 10;
    console.log(`    🔄 Rounded ${value.toFixed(2)} → ${rounded.toFixed(1)} (ends in .05)`);
    return rounded;
  }
  
  return value; // Иначе возвращаем без изменений
};
```

**Примеры работы:**
```javascript
roundUpIfNeeded(0.05)  → 0.1   ✅
roundUpIfNeeded(1.05)  → 1.1   ✅
roundUpIfNeeded(2.05)  → 2.1   ✅
roundUpIfNeeded(0.15)  → 0.15  ✅ (без изменений)
roundUpIfNeeded(1.25)  → 1.25  ✅ (без изменений)
roundUpIfNeeded(2.4)   → 2.4   ✅ (без изменений)
```
