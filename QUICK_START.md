## Тестирование PDF генератора (30 секунд)

### Шаг 1: Загрузка демо-данных
1. Откройте приложение
2. Нажмите кнопку **"Загрузить демо-данные для теста"** (вверху страницы)
3. Нажмите **"Продолжить"**

### Шаг 2: Настройка обложки
1. Измените текст обложки (опционально)
2. Выберите цвет фона
3. Загрузите изображение для обложки (опционально)
4. Нажмите **"Продолжить"**

### Шаг 3: Палитра
1. Просмотрите загруженные цвета (15 цветов DMC)
2. Нажмите **"Продолжить"**

### Шаг 4: Генерация PDF
1. Выберите опции PDF (все включены по умолчанию)
2. Нажмите **"Скачать PDF"**
3. ✅ Готово! PDF файл скачан!

## Что включено в сгенерированный PDF?

1. **Обложка** - С вашим дизайном
2. **Технический паспорт** - Размеры, канва, ткань
3. **Расход мулине** - Таблица DMC цветов
4. **Ключ символов** - Легенда символов
5. **Страницы схемы** - Placeholder для схемы

## Полезные кнопки

- **Сформировать PDF** - Генерирует PDF и показывает превью
- **Скачать PDF** - Мгновенно скачивает готовый файл
- **Открыть превью** - Открывает PDF в новой вкладке (после генерации)
