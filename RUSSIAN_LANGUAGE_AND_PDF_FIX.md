### 3. ✅ Добавлен выбор страниц для PDF

#### Новая функциональность выбора страниц
StepFive.tsx теперь содержит интерактивный список страниц с чекбоксами:

**Страницы, доступные для выбора:**
1. **Обложка** - всегда включена (обязательная)
2. **Технический паспорт** - можно включить/выключить
3. **Расход мулине** - можно включить/выключить
4. **Ключ символов** - можно включить/выключить

**Визуальные элементы:**
- Счетчик выбранных страниц: "Выбрано: X из 4"
- Подсказка: "Нажмите на чекбокс, чтобы включить или исключить страницу из итогового PDF"
- Иконки CheckSquare/Square для визуальной индикации
- Обложка помечена как "(обязательно)" и не может быть отключена

**Состояние по умолчанию:**
- Все страницы включены по умолчанию
- Пользователь может отключить любую страницу кроме обложки
- Выбор сохраняется в `projectData` через `updateProjectData`

Функция `generateAndDownloadPDF` принимает:

## Тестирование

### Как протестировать
1. Запустите приложение
2. Пройдите все шаги до Шага 5 "Генерация PDF"
3. Проверьте, что все тексты на русском языке
4. На шаге 5 попробуйте:
   - Отключить/включить разные страницы (кроме обложки)
   - Убедитесь, что счетчик обновляется
   - Нажмите "Скачать PDF"
   - Проверьте, что PDF сгенерирован и содержит только выбранные страницы

### Ожидаемый результат
- Все названия шагов на русском языке
- PDF генерируется с выбранными пользователем страницами
- Toast уведомления появляются при начале и завершении генерации
- При ошибке показывается соответствующее уведомление

## Известные ограничения

1. Обложка всегда включена в PDF (это обязательная страница)
2. Страницы схемы (pattern pages) пока содержат placeholder
3. PDF генерируется с базовыми настройками A4
