## 🔍 Как Работает

### Алгоритм:

```
1. Загрузка файла
   ↓
2. Парсинг строк
   ↓
3. Нормализация кодов DMC
   ├─ Распознавание склеенных
   ├─ Исправление неполных
   └─ Проверка корректных
   ↓
4. Группировка по кодам
   └─ Суммирование дубликатов
   ↓
5. Обработка комбинаций
   ├─ Деление длины пополам
   ├─ Добавление к компонентам
   └─ Удаление комбинаций
   ↓
6. Сортировка
   ↓
7. Нумерация
   ↓
8. Отображение в таблице
```

### Пример:

**Вход:**
```
666      2.2м   (обычный)
3799     4.0м   (обычный)
3103799  2.5м   (склеенные → 310+3799)
3+3799   0.8м   (неполный → 310+3799)
666+705  0.3м   (корректный)
```

**Обработка:**
```
310+3799: 2.5м + 0.8м = 3.3м
  ├─ 310:  3.3м ÷ 2 = 1.65м (новый)
  └─ 3799: 4.0м + 1.65м = 5.65м (обновлён)

666+705: 0.3м
  ├─ 666: 2.2м + 0.15м = 2.35м (обновлён)
  └─ 705: 0.15м (новый)
```

**Выход:**
```
№  DMC   Длина
1  310   1.65м
2  666   2.35м
3  705   0.15м
4  3799  5.65м
```
