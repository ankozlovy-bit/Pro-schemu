### Обработка Длины

- ✅ Длина всегда делится **пополам**
- ✅ Половины добавляются только к **первым двум** цветам
- ✅ Строки с символом `+` **удаляются** из финальной таблицы
- ✅ Автоматическое **объединение дубликатов**

## 🔍 Пример Работы

### Входной файл:
```
Номер цвета (DMC)	Наименование цвета	Общая длина (м)
666	Bright red	2.2
3799	Anthracite grey	4.0
3103799	-	2.5
3+3799	-	0.8
666+705	-	0.3
```

### Результат:
```
№  DMC   Название              Длина (м)  Расчёт
1  310   -                     1.65       (0 + 1.25 + 0.4)
2  666   Bright red            2.35       (2.2 + 0.15)
3  705   -                     0.15       (0 + 0.15)
4  3799  Anthracite grey       5.65       (4.0 + 1.25 + 0.4)
```

**Расшифровка:**
- `3103799` → `310+3799`: 2.5м ÷ 2 = по 1.25м каждому
- `3+3799` → `310+3799`: 0.8м ÷ 2 = по 0.4м каждому
- `666+705`: 0.3м ÷ 2 = по 0.15м каждому
- Все строки с `+` удалены
