# 🎨 StitchPDF Typography System 2026

Премиальная типографическая система на основе **100% бесплатных шрифтов** с лицензией SIL Open Font License.

## 📚 Шрифты

### 🎯 Primary: **Inter** (Sans-Serif)
- **Назначение**: Основной текст, UI элементы, параграфы
- **Характер**: Современный, технологичный, идеально читаемый
- **Источник**: Google Fonts
- **Лицензия**: SIL OFL 1.1
- **Использование**: `font-sans` или `var(--font-sans)`

### 👑 Secondary: **Playfair Display** (Serif)
- **Назначение**: Крупные заголовки, акценты, роскошь
- **Характер**: Элегантный, редакционный, высококонтрастный
- **Источник**: Google Fonts
- **Лицензия**: SIL OFL 1.1
- **Использование**: `font-serif` или `var(--font-serif)`

### ✨ Accent: **Plus Jakarta Sans** (Sans-Serif)
- **Назначение**: Кнопки, карточки, современные акценты
- **Характер**: Мягкий, геометричный, свежий
- **Источник**: Google Fonts
- **Лицензия**: SIL OFL 1.1
- **Использование**: `font-accent` или `var(--font-accent)`

### 💻 Mono: **JetBrains Mono** (Monospace)
- **Назначение**: Технические данные, коды, таблицы
- **Характер**: Точный, стильный, современный
- **Источник**: Google Fonts
- **Лицензия**: SIL OFL 1.1
- **Использование**: `font-mono` или `var(--font-mono)`

## 🎨 Когда использовать

### **Playfair Display** (Serif)
✅ Логотип "StitchPDF"  
✅ H1, H2 заголовки  
✅ Названия секций  
✅ Обложки PDF  
❌ Длинный текст  
❌ Мелкий кегль (<14px)

### **Inter** (Sans)
✅ Основной текст (body)  
✅ Описания, параграфы  
✅ Навигация  
✅ Формы (input, textarea)  
✅ Таблицы (основной текст)  
❌ Крупные декоративные заголовки

### **Plus Jakarta Sans** (Accent)
✅ Кнопки  
✅ Бейджи, теги  
✅ Карточки (заголовки)  
✅ Call-to-Action элементы  
✅ Подзаголовки  
❌ Длинный текст

### **JetBrains Mono** (Mono)
✅ Технические данные (размеры, коды)  
✅ Таблицы с числами  
✅ DMC коды нитей  
✅ Акцидентные элементы в дизайне  
❌ Основной текст  
❌ Декоративные заголовки

## 💡 Примеры использования

### В JSX/TSX:
```tsx
// С помощью Tailwind классов
<h1 className="font-serif text-4xl">Заголовок</h1>
<p className="font-sans text-base">Основной текст</p>
<button className="font-accent">Кнопка</button>
<code className="font-mono">DMC 310</code>

// С помощью CSS переменных
<div style={{ fontFamily: 'var(--font-serif)' }}>Текст</div>
```

### В CSS:
```css
.my-headline {
  font-family: var(--font-serif);
}

.my-body-text {
  font-family: var(--font-sans);
}

.my-button {
  font-family: var(--font-accent);
}

.my-code {
  font-family: var(--font-mono);
}
```

## 🚀 Почему эти шрифты?

1. **100% бесплатно** - никаких подписок, лицензий, ограничений
2. **Google Fonts** - быстрая загрузка через CDN
3. **Отличная читаемость** - протестированы миллионами пользователей
4. **Премиальный вид** - используются топовыми компаниями
5. **Поддержка кириллицы** - все шрифты поддерживают русский язык
6. **Variable fonts** - оптимизированы для веб-производительности

## 🔗 Альтернативы (если нужно изменить)

### Вместо Playfair Display:
- **Cormorant Garamond** - более тонкий, аристократичный
- **Lora** - мягче, для длинных текстов

- **Outfit** - более округлый
- **Sora** - технологичнее
