# ✅ Обновление v1.1.0 - ГОТОВО

## 🎉 Что было добавлено

### Автоматическое округление чисел с .05

Теперь при делении длины смешанных цветов пополам, если получается число, заканчивающееся на **.05**, оно автоматически округляется в **бОльшую сторону** (до **.1**).

---

## 📊 Примеры

### До обновления (v1.0.0):
```
Смешанный цвет: 0.1м
Деление: 0.1 ÷ 2 = 0.05м
Результат: 
  - 666: 1.5 + 0.05 = 1.55м
  - 703: 2.0 + 0.05 = 2.05м
```

### После обновления (v1.1.0):
```
Смешанный цвет: 0.1м
Деление: 0.1 ÷ 2 = 0.05м → округляем до 0.1м 🔄
Результат:
  - 666: 1.5 + 0.1 = 1.6м ✅
  - 703: 2.0 + 0.1 = 2.1м ✅
```

## 🔧 Технические детали

### Измененный файл:
`/src/app/components/ThreadCalculator.tsx`

### Добавлена функция:
```typescript
const roundUpIfNeeded = (value: number): number => {
  const decimalPart = value - Math.floor(value);
  const isPointZeroFive = Math.abs(decimalPart - 0.05) < 0.001;
  
  if (isPointZeroFive) {
    const rounded = Math.ceil(value * 10) / 10;
    console.log(`    🔄 Rounded ${value.toFixed(2)} → ${rounded.toFixed(1)} (ends in .05)`);
    return rounded;
  }
  return value;
};
```

### Применение:
```typescript
let lengthPerColor = row.lengthMeters / 2;
lengthPerColor = roundUpIfNeeded(lengthPerColor); // НОВОЕ
```
